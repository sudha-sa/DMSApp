# Document Management System (DMSApp)

A responsive mobile workspace application built using modern React Native with a full global state store layer mapping back cleanly to the expected structures.

## System Tech Stack Details

- **Architecture**: Core Native Community CLI template
- **State Engine**: Redux Toolkit (Slice authentication setup)
- **Navigation Flow**: React Native Native Stack & Bottom Tab Navigators

## Installation & Launching Sequence

1. Clone or download this repository folder hierarchy.
2. Install necessary project packages via the terminal root:

```bash
   npm install
```
